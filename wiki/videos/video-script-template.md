# 爆款视频脚本模板

> 3C类目 - 东南亚TikTok

## 🔥 黄金3秒法则

```
0-3秒: 痛点/效果展示 + 悬念
3-10秒: 产品介绍 + 核心卖点
10-25秒: 场景演示 + 使用效果
25-30秒: 引导互动 (关注/评论)
```

---

## 📝 爆款脚本结构

### 类型1: 开箱展示
```markdown
[画面: 快递盒特写 + 神秘感音效]
"这绝对是你见过最美的耳机开箱"
[画面: 缓慢打开包装]
"看这个光泽，这个质感..."
[画面: 产品全方位展示]
"连接手机试试..."
[画面: 测试降噪功能]
"这个降噪效果..."
[结尾: 上车引导]
"点击下方链接，限时优惠"
```

### 类型2: 对比测评
```markdown
[画面: 两款产品并排]
"花30块和300块买耳机有什么区别？"
[画面: 外观对比]
"外观上这个价位看不出差别..."
[画面: 音质测试]
"但音质差距还是明显的..."
[画面: 总结]
"我的建议是..."
```

### 类型3: 场景种草
```markdown
[画面: 通勤场景]
"每天挤地铁没有它不行"
[画面: 运动场景]
"跑步健身完全不会掉"
[画面: 办公场景]
"降噪一开 世界安静"
[画面: 颜值即正义]
"这个颜值 我给满分"
```
