## 二、macOS say局限性分析

| 问题 | 说明 | 影响 |
|------|------|------|
| 单一声音 | 只有Amira等标准声音 | 无法角色化 |
| 无语调控制 | 无法调整语速、音调 | 表达单一 |
| 无情感表达 | 无法表达愤怒、喜悦 | 缺乏感染力 |
| 无角色区分 | 所有人物声音相同 | 听不出差异 |

**结论：macOS say不适合水浒传角色化配音**

## 四、Sambert角色化配置

### 武松声音配置

```json
{
  "role": "武松",
  "voice_type": "zhitian_emo",  // 武侠男声
  "speech_rate": -0.2,          // 语速减慢20%
  "pitch_rate": 0.15,           // 音调提高15%
  "emotion": "heroic",          // 英勇情感
  "style": "ancient_chivalrous" // 古风侠义
}
```

### 店小二声音配置

```json
{
  "role": "店小二",
  "voice_type": "xiaoyu_emo",   // 小人物声
  "speech_rate": 0.3,           // 语速加快30%
  "pitch_rate": -0.1,           // 音调降低10%
  "emotion": "servile",         // 谦卑情感
  "style": "obsequious"         // 阿谀奉承
}
```

### 老虎声音配置（特效）

```json
{
  "role": "老虎",
  "voice_type": "roar_effect",  // 咆哮特效
  "speech_rate": -0.5,          // 语速极慢
  "pitch_rate": -0.3,           // 音调极低
  "effect": "tiger_roar"        // 老虎咆哮音效
}
```
