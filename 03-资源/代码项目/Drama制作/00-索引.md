# Drama制作代码项目

## 1. water-margin-drama (水浒传AI短剧)
**位置**: `~/.openclaw/skills/water-margin-drama/`

### 功能
- 水浒传120回短剧自动生成
- 角色设计 (108好汉)
- 争议检测与改写
- 视频合成 + 配音

### 核心文件
| 文件 | 用途 |
|------|------|
| water_margin_drama.py | 主程序 |
| drama_script.py | 剧本生成 |
| drama_voice.py | 语音合成 |
| drama_video.py | 视频生成 |
| controversy_rewriter.py | 争议改写 |

### 运行
```bash
cd ~/.openclaw/skills/water-margin-drama
python3 water_margin_drama.py --episode 1
```
