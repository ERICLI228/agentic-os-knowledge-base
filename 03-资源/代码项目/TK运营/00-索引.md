# TK运营代码项目

## proactive-operator
**位置**: `~/.agents/skills/proactive-operator/`

### 功能
- 监控TikTok东南亚5国3C品类
- 16个品类实时监控
- 爆款告警 (300万播放阈值)

### 目录结构
```
proactive-operator/
├── proactive-operator.sh    # 主脚本
├── data/
│   ├── tiktok_watch_*.json  # 原始数据
│   └── report_*.md          # 报告
└── config/
    └── categories.yaml      # 品类配置
```

### 运行
```bash
cd ~/.agents/skills/proactive-operator
bash proactive-operator.sh
```

### Cron: 每4小时
