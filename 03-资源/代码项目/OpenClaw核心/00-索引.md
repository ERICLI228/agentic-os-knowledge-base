# OpenClaw核心代码

## 主要目录

| 目录 | 内容 |
|------|------|
| `~/.openclaw/core/` | 核心API |
| `~/.openclaw/skills/` | 200+技能 |
| `~/.openclaw/scripts/` | 自动化脚本 |
| `~/.agents/skills/` | Agent技能 |

## 关键文件

| 文件 | 用途 |
|------|------|
| `~/.openclaw/openclaw.json` | 主配置 |
| `~/.openclaw/core/dashboard_api_v2.py` | Dashboard API |
| `~/.openclaw/core/task_helper.py` | 任务处理 |

## 查看命令

```bash
# 状态
openclaw status

# 技能列表
openclaw skills list

# 任务
openclaw tasks list
```
